# 住宅ローン金利比較アプリ 仕様書

**バージョン**: 1.0  
**作成日**: 2026-06-01  

---

## 1. プロダクト概要

### 1.1 アプリ名（仮）
住宅ローン金利ナビ

### 1.2 目的
住宅購入を検討している個人が、各銀行の最新の住宅ローン金利を一か所で比較・確認し、返済シミュレーションを行えるようにする。

### 1.3 ターゲットユーザー
住宅購入を検討中の一般個人（ローン初心者を含む）

## 5. 機能要件

### 5.1 金利一覧・比較画面（コア機能）

| 機能 | 詳細 |
|------|------|
| 一覧表示 | 各銀行の変動金利を一覧でテーブル表示 |
| ソート | 金利の低い順・高い順で並び替え |
| フィルター | 銀行カテゴリ（メガバンク／ネット銀行等）で絞り込み |
| 最終更新日時 | 各銀行の金利データの最終取得日時を表示 |

**表示カラム例**
```
銀行名 | 変動金利（年） | キャンペーン | 審査目安 | 更新日時 | 詳細リンク
```

### 5.2 銀行詳細画面

| 項目 | 内容 |
|------|------|
| 基本金利 | 変動金利（通常・優遇後） |
| 審査基準・条件 | 最低年収目安・雇用形態・その他主な条件 |
| キャンペーン・特典 | 現在実施中のキャンペーン情報（期限付き） |
| 公式サイトリンク | 住宅ローン申込ページへの直接リンク |
| 最終更新日時 | スクレイピングで取得した日時 |

### 5.3 返済シミュレーション機能（コア機能）

ユーザーが以下を入力し、月々の返済額を計算・比較できる。

**入力項目**
| 項目 | 備考 |
|------|------|
| 借入額 | 万円単位で入力 |
| 借入期間 | 最大35年、1年刻み |
| 返済方式 | 元利均等返済（デフォルト） |

**出力項目**
| 項目 | 備考 |
|------|------|
| 月々の返済額 | 全銀行分を一覧で並べて比較 |
| 総返済額 | 銀行ごとに表示 |
| 銀行間の差額 | 最安値との比較額を表示 |

**計算式（元利均等返済）**
```
月利 r = 年利 / 12
月返済額 = 借入額 × r × (1+r)^n / ((1+r)^n - 1)
※ n = 返済月数
```

## 6. 非機能要件

### 6.1 データ取得
| 項目 | 内容 |
|------|------|
| 取得方法 | Webスクレイピング（各銀行の公式サイトから自動取得） |
| 取得頻度 | 月1回（金利は基本月次で変更されるため） |
| データ保存 | DBに保存し、アプリはDBから読み取る |
| 異常検知 | スクレイピング失敗時はアラートを発報し、前回データを表示 |

### 6.2 パフォーマンス
- 初回表示：3秒以内
- シミュレーション計算：即時（クライアントサイドで実行）

### 6.3 PWA要件
| 項目 | 内容 |
|------|------|
| ホーム画面追加 | iOS・Android対応のmanifest.json設定 |
| オフライン | 最後に取得したデータをキャッシュして参照可能 |
| 通知（将来） | 金利変動時のプッシュ通知（v2候補） |

### 6.4 セキュリティ・法的配慮
- 各銀行の利用規約・robots.txtを確認し、スクレイピング可否を事前確認
- 表示するデータは参考情報である旨を明示（免責事項を掲載）
- 個人情報は収集しない（シミュレーション入力値はサーバー送信しない）


## 8. 技術スタック（候補）

| レイヤー | 候補技術 | 備考 |
|----------|----------|------|
| フロントエンド | Next.js (React) | SSG/SSRでSEOも対応可 |
| スタイリング | Tailwind CSS | レスポンシブ対応が容易 |
| PWA | next-pwa | Service Worker自動生成 |
| バックエンド | Node.js / Python | スクレイピング処理 |
| スクレイピング | Playwright / Puppeteer | JS描画ページにも対応 |
| DB | PostgreSQL / SQLite | 金利データ保存 |
| ホスティング | Vercel / Railway | フロント・バックを分離または統合 |
| スケジューラー | Cron / GitHub Actions | 月次スクレイピング自動実行 |
